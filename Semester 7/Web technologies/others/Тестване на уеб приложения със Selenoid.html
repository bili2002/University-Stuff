Тестване на уеб приложения със Selenoid

Тестването на уеб приложения е ключова част от съвременното разработване на софтуер. Selenoid е модерен инструмент, който предоставя бърз и надежден начин за автоматизация на тестовете с използване на контейнери.

Основни концепции на Selenoid

  • Работа с браузъри в контейнери чрез Docker.
  • Поддръжка на VNC за визуализиране на тестовете.
  • Мащабируемост и по-добра производителност спрямо Selenium Grid.

Инсталация и конфигурация

Следвайте стъпките по-долу, за да инсталирате Selenoid:

docker pull selenoid/vnc:chrome_100.0
docker run -d --name selenoid -p 4444:4444 selenoid/vnc:chrome_100.0

Примери за тестване със Selenoid

Ето примерен код за автоматизиран тест на Python:

from selenium import webdriver

capabilities = {
"browserName": "chrome",
"browserVersion": "100.0",
"selenoid:options": {
"enableVNC": True,
"enableVideo": False
}
}

driver = webdriver.Remote(
command_executor='http://localhost:4444/wd/hub',
desired_capabilities=capabilities
)
driver.get("https://example.com")
print(driver.title)
driver.quit()
